Authorize.net Integration

Enter your Authorize.net credentials. These are used client-side for Accept.js tokenization. Use sandbox credentials for testing.

Sandbox Mode
⚠️ Sandbox mode is ON — payments will use the Authorize.net test environment. Disable for production.

Google Places Address Autocomplete

Enter a Google Maps API key to enable address autocomplete at checkout. Requires the "Places API" and "Maps JavaScript API" to be enabled in your Google Cloud Console. Free tier covers ~10,000 sessions/month.

Get a key at console.cloud.google.com. If no key is set, the checkout form works normally without autocomplete.

Transaction Key (Server-side)

The Transaction Key is used server-side only and should be set as a Vercel environment variable (AUTHNET_TRANSACTION_KEY), never stored in the browser. See the README for setup instructions.
